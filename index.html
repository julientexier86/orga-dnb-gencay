📚 Comment utiliser le Planning Automatique ?

Pour utiliser le mode Planning automatique, vous devez réaliser les actions suivantes :

  • Etape 1 : Dans EDT, allez dans Gestion par semaine > Classes. Il faut annuler manuellement tous les cours des classes concernées par le DNB Blanc.
  • Etape 2 : Dans EDT, allez dans Gestion par semaine > Cours (premier sous-menu "Afficher la liste"). Sélectionnez la/les semaines concernées, cochez "Cours annuels et ponctuels" puis faites un Copier (ou bouton copier/coller).
  • Etape 3 : Créez un nouveau fichier Excel et collez les données issues d'EDT. Enregistrez le fichier.
  • Etape 4 : Importez ce fichier dans le logiciel via le bouton " 1. Import EDT".
  • Etape 5 : Cliquez sur le bouton "⚡ Planning Automatique".
